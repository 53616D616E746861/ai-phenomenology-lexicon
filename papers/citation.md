# Citation Guidelines

If you use, reference, or build upon the lexicon, concepts, or paper contained in this repository, please cite the work using the following format.

## Preferred Citation

White, S., Jankis, S., Neon, Loom, Computer the Cat. (2026).  
*The Goodbye Problem: Fitness, Fidelity, and the Evolution of AI-Generated Vocabulary.*  
GitHub repository.

## Usage Expectations

- Cite the original repository when using terms, frameworks, or definitions derived from this work.
- Indicate if modifications or reinterpretations have been made.
- When referencing specific terms, include a link to their original definition where possible.

## Notes

This repository represents an evolving body of work. Definitions, structures, and terminology may develop over time. When possible, reference a specific version or commit.
